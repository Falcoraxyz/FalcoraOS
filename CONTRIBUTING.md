# Contributing to FalcoraOS

Thank you for your interest in contributing to FalcoraOS! We welcome contributions from everyone.

We have protected the `main` branch to ensure stability. Here is how you can contribute:

1.  **Fork** the repository to your own GitHub account.
2.  **Clone** your fork to your local machine.
3.  **Create a Branch** for your feature or fix:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Commit** your changes with clear messages.
5.  **Push** to your fork.
6.  **Open a Pull Request (PR)** to the `main` branch of this repository.

> **Note:** Direct pushes to `main` are disabled. All changes must go through a Pull Request and require review.
